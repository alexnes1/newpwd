# newpwd - password generator
    $ newpwd
    )r6GN]_?i|

## Installation
`go install -v github.com/alexnes1/newpwd/cmd/newpwd@v1`

Or download an executable file from "Releases" section.

## Options
### Password length
    $ newpwd --length 15
    $ # or
    $ newpwd -l 15

### No puctuation symbols
    $ newpwd --no-punc
    $ # or
    $ newpwd -p

### No digits
    $ newpwd --no-digits
    $ # or
    $ newpwd -d

### No uppercase letters 
    $ newpwd --no-upper
    $ # or
    $ newpwd -u


### No lowercase letters
    $ newpwd --no-lower
    $ # or
    $ newpwd -w
